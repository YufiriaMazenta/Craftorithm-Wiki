# 基础

## 简介

有时，我们希望玩家在合成某个配方时需要满足某些条件，或玩家在合成某个配方后执行某些操作。而插件提供的动作系统可以在玩家合成工作台和锻造配方时进行条件判断和动作执行。

## 语句

插件定义语句有条件判断语句、动作执行语句和转向语句。每个语句都会有自己的返回值，而返回值可以被某些特殊的关键词二次使用。
